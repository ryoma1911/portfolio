# Ryoma Abe - Portfolio

![Portfolio Screenshot](./public/ogp.png) <!-- 適宜画像パスを修正 -->

## 概要

1996年生まれのITエンジニア「アオベ」によるポートフォリオサイトです。これまでのキャリア・スキル・制作物を紹介しています。

本ポートフォリオは、React.js をベースに開発され、AWS にデプロイされています。

---

## 🔧 使用技術

- フロントエンド: `React.js`, `Vite`, `Tailwind CSS`
- アニメーション: `Framer Motion`
- アイコン: `React Icons`
- バージョン管理: `GitHub`
- インフラ: `AWS (S3 / CloudFront など予定)`

---

## 📂 ディレクトリ構成
```
src/
├── components/ # 各セクション（Header, About, Skills, Works, Contact, Careerなど）
├── assets/ # 画像やアイコン
├── App.jsx # アプリのルーティングと全体構成
└── index.jsx # エントリポイント
```
## 🔥 特徴

- 自己紹介やスキル・制作実績などを分かりやすく構成
- カード形式の作品紹介
- Careerセクションで業務・個人開発歴を時系列で整理
- アニメーションで動きのあるデザイン
- 今後はブログ機能や日本語・英語切り替え対応も検討中
